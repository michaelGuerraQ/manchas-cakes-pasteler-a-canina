# 🎂 manchas-cakes

Catálogo web de tortas personalizadas desarrollado con **React + TypeScript** para mostrar productos de forma visual, moderna y atractiva, facilitando el contacto con clientes y la gestión de pedidos mediante **WhatsApp**.

---

## ✨ Funcionalidades

- 🛍️ **Catálogo visual de productos**  
  Muestra tortas personalizadas de forma ordenada, atractiva y fácil de explorar.

- 📱 **Diseño responsive**  
  Adaptado para dispositivos móviles, tablets y computadoras.

- 🔗 **Integración con WhatsApp**  
  Permite que los clientes se contacten rápidamente para realizar pedidos personalizados.

- ⚡ **Navegación fluida**  
  Interfaz rápida, moderna y enfocada en una buena experiencia de usuario.

- 🎨 **Diseño orientado a ventas**  
  Presentación visual pensada para un emprendimiento real de repostería.

- 🧩 **Estructura escalable**  
  Proyecto organizado para poder agregar nuevas funcionalidades en el futuro.
