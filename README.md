## 📌 Descripción

**Manchas Cakes** es un catálogo web diseñado para mostrar tortas personalizadas de forma visual, moderna y atractiva, facilitando el contacto directo con clientes a través de WhatsApp.

Este proyecto está orientado a un **emprendimiento real**, enfocado en mejorar la experiencia del cliente y aumentar las conversiones mediante una interfaz clara y llamativa.

---

- Fácil exploración de productos

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/michaelGuerraQ/manchas-cakes.git

# Entrar al proyecto
cd manchas-cakes

# Instalar dependencias
npm install
```

---

## ▶️ Ejecución

```bash
npm run dev
```
